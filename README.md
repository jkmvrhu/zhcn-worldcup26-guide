# zhcn-worldcup26-guide

本仓库用于归档和发布多个独立的 HTML 页面，旨在整理与 **2026 年国际足联世界杯**（FIFA World Cup 2026）相关的中文信息与参考资源。

## 项目简介

这是一个静态文档与页面集合仓库，不绑定任何特定网站或域名。所有页面均为独立 HTML 文件，方便直接查看、分享或嵌入。内容主要围绕 2026 年世界杯的赛程、队伍、场馆、历史数据等公开信息进行整理，供感兴趣的用户自行浏览或参考。

## 目录结构

```
.
├── README.md          # 本文件
├── index.html         # 首页/导航页（可选）
├── pages/             # 存放所有独立 HTML 页面
│   ├── schedule.html  # 赛程相关页面
│   ├── teams.html     # 球队信息页面
│   ├── venues.html    # 场馆介绍页面
│   └── ...
├── assets/            # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── docs/              # 额外文档或说明（可选）
```

> 注：具体文件以仓库实际内容为准，上述结构仅为示例。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面之间可能存在链接跳转，但整体结构为扁平化归档，不依赖后台或数据库。
- 所有内容均基于公开可获取的信息整理，不包含任何商业推广或诱导点击元素。
- 页面样式力求简洁、清晰，适合快速查阅。
